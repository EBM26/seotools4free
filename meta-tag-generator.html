Tool 04 / SEO

Meta Tag Generator

Fill in your page details to generate title, meta description, canonical, Open Graph, and Twitter Card tags ready to paste into your page's <head>.

Shows in the browser tab and as the link title in search results.

Shows below the title in search results. Aim for under 160 characters.

Tells search engines the preferred URL for this content, prevents duplicate-content issues.

Used for both Open Graph and Twitter Card previews. Recommended: 1200×630px.

Shows next to the preview on some platforms (e.g. Facebook).

Attributes the card to a Twitter/X account, if you have one.

What each tag does

Tag Where it shows up
Title tag
<title>		 Browser tabs, bookmarks, and as the clickable headline in search results.
Meta description
meta[name=description]		 The snippet text below the title in search results. Doesn't affect ranking directly, but affects click-through.
Canonical link
link[rel=canonical]		 Not visible to users, tells search engines which URL is the "real" one when the same content is reachable at more than one address.
Open Graph tags
og:*		 Controls how the page looks when shared as a link card on Facebook, LinkedIn, Slack, and most other platforms.
Twitter Card tags
twitter:*		 Same idea as Open Graph, but specific to X/Twitter's link preview format.
